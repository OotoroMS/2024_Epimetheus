# Feature Directory Layout

This project now groups assets by feature under `Assets/Features/`.

- `Battle/`: migrated from `Assets/Test/BattleSystem`
- `Dungeon/`: migrated from `Assets/Test/DungeonSystem`
- `Title/`: migrated from `Assets/0_Title`

Each feature is organized with the following purpose folders:

- `Code/`
- `Content/`
- `Data/`
- `Scenes/`
- `Docs/`
