# Title Feature README

## Responsibility
- タイトル画面の表示と、ゲーム開始前の導線（開始/設定/終了）を担当する。
- タイトルBGMや初期演出の再生開始・停止を管理する。
- タイトルから次シーンへの遷移要求を受け付け、遷移処理を開始する。

## Public API / Entry points
- アセットエントリ: `Assets/0_Title/`（タイトル関連リソース）。
- BGMアセット: `Assets/0_Title/Music/Title.mp3`。
- （将来）TitleシーンのルートコンポーネントをFeatureの単一入口として定義する。

## Invariants
- タイトル表示中はゲーム本編ロジックを開始しないこと。
- タイトルBGMは重複再生しないこと。
- 開始操作は多重実行されず、シーン遷移要求は1回のみ受理すること。
- どの遷移経路でもタイトルの一時オブジェクトをリークさせないこと。

## Test cases
### EditMode
- タイトル状態管理: 初期状態が `Idle/Ready` などの想定状態であること。
- 遷移要求の多重入力防止: 連続呼び出しで1回だけ有効化されること。
- オーディオ管理: BGM再生要求の重複抑止が機能すること。

### PlayMode
- タイトル起動時にUIとBGMが正しく初期化されること。
- 開始ボタン押下で次シーンへ遷移すること。
- 設定/終了ボタン押下時に想定イベント（画面遷移/終了処理）が呼ばれること。
