# 戦闘フロー

1. 初期化と初回メッセージ表示
2. メインループ開始(無限ループ)
  1. プレイヤーコマンド受付ループ(人数回数繰り返す。変数:i)
    1. i番目のプレイヤーのコマンド入力を受け付け
    2. iをインクリメント
  2. 敵味方共通で素早さ順に行動決定
  3. 戦闘行動ループ(敵、味方がすべて動くまで繰り返す)
    1. 戦闘行動処理
      1. もし攻撃対象のHPが0を下回ったら、その対象は死亡する。この時点ではデータを削除しない。
        1. もし対象が死亡したことで、敵、もしくはパーティーが全滅したかチェック
  4. コマンド情報を初期化する。
